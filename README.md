### Rules

- 문제 선정 : 매주 금요일
- 라이브 코딩 : 매주 월요일, 목요일 23:00 ~ 24:00 (필참X)
- 문제 리뷰 : 매주 화요일, 금요일 22:00까지 업로드
  - 리뷰 내용) 문제 접근 방식 + 코드 파일
  - 다른 사람의 리뷰에 대해 `comment` / `approved`
<br><br>

### Convention Rule

**Issue** | `[사이트명] 문제 번호 - 문제 이름`  
ex) [백준] 1158 - 요세푸스 문제

**PR** | `[#이슈번호] 이름(한글)`  
ex) [#109] 송연

**Commit** | `커밋내용 문제이름 파일명`<br> 
ex) Create 요세푸스 문제 ssong.java
